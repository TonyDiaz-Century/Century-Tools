220.82 — Dwelling Optional

For a single dwelling unit — new or existing — served by a 120/240V 3-wire service. Most common method for service sizing on residential projects.

Use when: sizing a service/feeder for an entire home, especially with EVSE, heat pumps, or mixed heating/cooling.

220.83(A) — No Additional Load

For an existing dwelling where no new load is being added — you're verifying whether the existing service is sufficient as-is.

Use when: confirming a service can carry the loads it already has (e.g., after a remodel that doesn't add load).

220.83(B) — Additional Load Added

For an existing dwelling where additional A/C or electric space-heating is being added. Applies stricter factors to the new load.

Use when: adding A/C, heat pump, or electric heat to an existing home without a service upgrade.

220.42 — Standard Method

The long-form calculation using individual demand factors per load type (general lighting, fastened-in-place appliances, range, dryer, etc.).

Use when: the optional methods don't apply, or the AHJ requires the standard calc.

EVSE Service-Capacity Check

Specifically for adding an EV charger to an existing service. Wraps 220.82 with EVSE at 100% per 625.42.

Use when: the main question is "will the existing service support this EVSE?"

220.83(B) — Existing Dwelling, Additional A/C or Heat Added
Use when additional A/C or electric space-heating is being added to an existing dwelling without a service upgrade. The calculation uses stricter rules for the new heating/cooling load.
220.83(B) rules:
• New A/C and heat-pump compressor/condenser: 100% of nameplate.
• New electric space-heating (including heat-pump strips): 100% of nameplate.
• Existing loads (lighting, appliances, existing A/C or heat) apply the 220.83 factors: first 8 kVA @ 100%, remainder @ 40%.
• Use the larger of (existing A/C) vs (existing space-heat) for the "existing" side. New additions are added on top.

220.42 — Standard (Long-Form) Method
The full standard calculation using individual demand factors per Article 220 Parts I–IV. Each load category uses its own factor: general lighting per Table 220.42(A), appliances per 220.53, range per 220.55 / Table 220.55, dryer per 220.54.
1. Service & Dwelling
General lighting + SA + laundry get Table 220.42(A) demand: first 3,000 VA @ 100%, 3,001–120,000 @ 35%, over 120,000 @ 25%.
2. Fastened-in-Place Appliances (220.53)
Excludes range, dryer, A/C, and space-heat (those have their own rules). If there are 4 or more fastened-in-place appliances on the same feeder, demand factor of 75% applies to their total.
3. Electric Range / Cooking (220.55)
Enter 0 if no electric range. Uses Column C demand: 8,000 VA for one appliance ≤12kW.
4. Clothes Dryer (220.54)
Minimum 5,000 VA if present. One dryer = 100%.
5. Heating / Cooling (220.60)
Largest of the two is used (non-coincident).
6. EVSE (optional, 625.42)
100% — no demand factor
